## Support

### Where to get help

- **Questions / help**: Open a GitHub Issue using the **Agent Support Request**
  template.
- **Bugs**: Use the **Bug Report** template and include:
  - your agent + version
  - Agent Response Boxes version
  - install mode (basic/enhanced/full) and scope (user/project)
  - any relevant logs/output (redact secrets)
- **Feature ideas**: Use the **Feature Request** template.

Response Boxes stores a local event log at `~/.response-boxes/analytics/boxes.jsonl`.
This may contain code snippets and repository metadata. Review `SECURITY.md` for
